使用注意事项：
1. 在src下建立db.properties
2. 每张表只能有一个主键。不能处理多个主键的情况。
3. pojo使用包装类，不要使用基本数据类型。



使用方法：
1、编缉src中的db.properties文件，连接至数据库。
2、执行cn.ne.orm.utils.JavaFileUtils中的main()方法或调用instancePOJO()方法，生成数据表对应的POJO。
3、创建Query接口实例，即可调用其中生成的CRUD方法。
4、目前只提供基础的查询，如需复杂查询，自行编写SQL并编缉自动生成的对应POJO类，即可执行关联查询。
